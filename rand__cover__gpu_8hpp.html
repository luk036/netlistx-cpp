Detailed Description

GPU-Accelerated Randomized Vertex Cover using Pitt's Algorithm.

Ported from Python rand_cover_gpu.py and Rust rand_cover_gpu.rs.

Runs multiple independent Pitt trials in parallel on the GPU via CUDA. Each CUDA thread executes one complete Pitt trial (edge iteration + random vertex selection). Falls back to CPU multi-threaded execution when CUDA is unavailable.

Note
Compile with -DHAS_CUDA to enable GPU acceleration. Requires CUDA toolkit and nvcc.

Reference: L. Pitt, "A Simple Probabilistic Approximation Algorithm for Vertex Cover," Technical Report, Yale University, 1985.
